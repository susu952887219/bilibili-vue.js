# vue-bilibili
哔哩哔哩移动端开发，主要功能有权限验证、筛选视频分类、用户上传头像、编辑个人信息、视频播放、推荐文章、收藏文章 、关注用户、盖楼评论、发表评论、登录与注册模块。使用的技术栈为 vue + vuex + vue-router + ES6+ less + flex + vant。
## Project setup
```
npm install
```
